# CLAUDE.md

## コマンド

### テスト

```bash
npm test
```

### リンター

```bash
npm run lint
```

### フォーマッター

```bash
npm run format
```

### セットアップ

```bash
npm install
```

依存パッケージをインストールする。

## 設計仕様（ADR）

設計仕様は `doc/adr/` に連番で管理される（`adr_001.md`, `adr_002.md`, ...）。
最新の仕様は最も大きい連番のファイルに記載されている。

## 実装手順

ADRに従って実装せよ。毎回実装した結果はREADME.mdに反映させること。
また、実装を行うごとに下記のテスト、リンター、フォーマッターを実行して確認せよ。
